# tfg_faces

Siguiendo el libro _Matrix methods in data mining_ intentamos hacer un modelo de reconocimiento facial con tensores.

La idea es conseguir un set de entrenamiento en el que un conjunto de personas aparezca con distintas expresiones faciales y nuestro trabajo sería, dada una nueva imagen de una de estas personas, ser capaz de identificar quién es.

Usamos una carpeta llamada datasets/FEI_face_database/combined/ con todas las fotos del dataset.
